Remote Control Dirigible Principles of Engineering

Overview

In this project, I worked on a four-person team to design and fabricate a remote control dirigible. My technical focus was on the electrical and software side of the project, particularly designing the wireless communication protocol, and designing/assembling the electrical subsystem. I also served as the project manager for this endeavor.

Keywords

circuit design, electrical prototyping, embedded devices, communication protocol design, wireless communication, project management

Course Description

Principles of Engineering is meant to be an introduction to truly multidisciplinary engineering projects. The course consists of a few short labs focused on concepts from different fields, then culminates in a major project. The final project is very open ended, and has few requirements: a $250 budget, and significant electrical, software, and mechanical components. For our project, we decided to build a remote control dirigible.

Project Description

I had two primary roles on this team: designing the communication protocol between the dirigible and its base station, and developing the electronics package to drive the system. Both of these tasks were far outside my skillset at the time, but I sought to broaden my toolset by taking on these challenges.

Communication Protocol

For an extensive description of this system, please visit the project site for further details. For a short description, please keep reading! Our chosen wireless solution for the project was an XBee series 1 module, which effectively works as a wireless serial port. This is a very simple, fairly low-level communication protocol which provides few tools to organize communication between devices. My work focused on devising a 'handshake' method to synchronize communications between the dirigible and the base station, which ultimately allowed for multiplexing data, synchronizing messages, and detecting packet errors.

Electronics Package

Again, for an extensive description of this system, please see this website. The electrical subsystem for the dirigible is fairly straightforward. The system is based around an Arduino Pro Mini, and serves to regulate the input voltage from the battery, and interface the Arduino with two servos and two DC motors driving propellers. I developed the system by first assembling the system components in a piecewise fashion, eventually assembling a 'roadkill' functional mockup of the entire system. Once all components were functioning properly, I designed a permanent system and soldered together components on perforated prototyping board.

Project Management

In order to gain manangerial experience, I decided to serve as the Project Manager (PM) for my team. My role as PM included identifying the critical path to project completion, breaking that path down into manageable chunks of work and clear milestones, and tracking progress towards those goals. I tried to maintain a hands-off approach to PM'ing, electing to meerly track progress and avoid working on tasks assigned to other people. Ultimately, we finished our minimum deliverable a week ahead of schedule.
